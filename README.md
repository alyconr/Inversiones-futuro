
# Inversiones Futuro - Landing Page Premium

Landing page inmobiliaria moderna, minimalista y corporativa construida con **React**, **Three.js** y **Tailwind CSS**.

## Características principales

- **Hero 3D:** Escena interactiva con Three.js (@react-three/fiber).
- **Modo Oscuro/Claro:** Implementado con Tailwind `class` y persistencia en `localStorage`.
- **Diseño Responsivo:** Mobile-first, optimizado para todos los dispositivos.
- **Animaciones:** Framer Motion para transiciones suaves y reveals de scroll.
- **Formulario de Contacto:** Con validaciones y estados de carga.
- **Sección Legal:** Política de privacidad adaptada al contexto colombiano.

## Tecnologías

- React 18+
- TypeScript
- Vite
- Tailwind CSS
- Three.js & React-Three-Fiber
- Framer Motion
- Lucide React

## Pasos para instalar y correr

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Ejecutar en desarrollo:**
   ```bash
   npm run dev
   ```

3. **Construir para producción:**
   ```bash
   npm run build
   ```

## Notas de Diseño

- La paleta de colores utiliza un azul profundo (`corporate-blue`) y acentos dorados suaves para transmitir confianza y exclusividad.
- El canvas 3D está optimizado para el rendimiento utilizando geometrías simples y un ciclo de renderizado eficiente.
